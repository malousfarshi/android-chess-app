# Android Chess App

Android chess application built with Kotlin and Jetpack Compose.  
The app is fully playable offline and allows players to enjoy a complete chess game experience.

## Features
- User registration and login (without Gmail integration yet)
- Offline gameplay
- Settings to customize colors and fonts
- Move history shows all moves performed during the game
- Fully developed game logic (90% complete)
- Can be extended with a database for online features
  
## Technical Details
- Built with Kotlin and Jetpack Compose
- Navigation handled via NavController
- Theme customization managed with ThemeViewModel
- Move history implemented to track all moves

- Project is under development
